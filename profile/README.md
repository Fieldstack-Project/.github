# Fieldstack Project

> Open-source modular productivity framework for personal and small team use

**Developed and maintained by [PSquare DIVISION](https://github.com/psquare-division)**

---

## 🎯 What is Fieldstack?

Fieldstack is a self-hosted, modular framework that helps you manage your personal finances, tasks, projects, and productivity—all in one place. Built on the principle of complete user control and privacy.

### Core Philosophy

- **🔓 Completely Free** - No premium features, no paywalls
- **🏠 Self-hosted First** - Your data stays on your server
- **🧩 Modular Design** - Install only what you need
- **🌐 Open Source** - MIT License, fully transparent
- **🤝 Community-driven** - Built by users, for users

## 🌟 Key Features

### Framework
- 🔌 **Plugin Architecture** - WordPress/VSCode-like extensibility
- 🏪 **Module Marketplace** - Discover and install modules via web UI
- 🤖 **AI Integration** - Support for Gemini, OpenAI, Claude, Ollama
- 🔄 **Auto Updates** - Scheduled automatic updates
- 🌐 **Multi-language** - Korean, English, and more

### Built-in Capabilities
- 💰 Finance & budget tracking
- 📅 Subscription management with calendar sync
- ✅ Task and project management
- 📊 Data visualization and reporting
- 🔐 Google Workspace integration
- 🔒 Privacy-first design

---

## 💻 Tech Stack

- **Backend**: TypeScript + Node.js + Express/Fastify
- **Frontend**: React + TypeScript + Vite
- **Database**: PostgreSQL / SQLite / Supabase / MongoDB / and more...
- **Deployment**: Docker / Railway(Docker) / Cloudflare
- **Monorepo**: pnpm workspaces
